# おもろい三麻 ミクボイス 元データ

三麻の宣言ボイス4種を初音ミク(V4X)で作るための元データです。

## ファイル一覧 (対応表)

| 元データ | フレーズ | 音程イメージ | 納品ファイル名 | ゲーム内の用途 |
|---|---|---|---|---|
| `riichi.vsqx` / `.mid` | リーチ! | り(D5)→ち(A4) 強めに落とす | `riichi.wav` | リーチ宣言時 |
| `ron.vsqx` / `.mid` | ロン! | ろ(D5)→ん(A4) | `ron.wav` | ロン和了時 |
| `tsumo.vsqx` / `.mid` | ツモ! | つ(B4)→も(D5) 尻上がりに元気よく | `tsumo.wav` | ツモ和了時 |
| `kita.vsqx` / `.mid` | ペイ!(北) | ぺ(D5)→い(A4) | `kita.wav` | 北抜き時 |

- コード側の参照: `script.js` の `VOICE_DEFS` が `assets/voice/{riichi|ron|tsumo|kita}.mp3 → .wav → 内蔵音声` の順で探します(**ファイルを置くだけで差し替わる**、コード変更不要)
- ストーリー使用: ✅ ゲーム本編の宣言演出(カットイン+SEと同時再生)

## 作り方 (Piapro Studio + Cubase)

1. Cubase で Piapro Studio を立ち上げ、**ファイル → インポート → `riichi.vsqx`**
   - シンガーの割当てを聞かれたら **ミク(V4X)** を選択
   - 万一 VSQX が読めない場合は同名の `.mid` をインポート → ノートを全選択して歌詞流し込みに「りち」等を入力
2. 発音・音程・長さはあくまで**叩き台**です。ミクらしく聞こえるよう自由に調整してください(BPM120、各フレーズ0.6〜0.8秒想定。頭の無音は詰まっているほど良い)
3. **書き出し**: Cubase「ファイル → 書き出し → オーディオミックスダウン」で WAV(44.1kHz/16bit)
   ※ Piapro 単体ではオーディオ書き出し不可(以前のセットアップメモ通り)
4. ファイル名を上の表の**納品ファイル名**どおりにして、この隣の
   `claude/omoroi-sanma/voice/` フォルダ(無ければ作成)に置く
5. Claude に「ミクボイス置いたよ」と一言 → repo に組み込み+クレジット表記追加+リリースします

## メモ

- 4フレーズとも 2ノート構成のシンプルな掛け声です。ビブラートはオフにしてあります
- 音量はなるべく4ファイルで揃えてください(バラつくとゲーム内で音量差が出ます)
- ライセンス: 非営利公開アプリなのでピアプロ・キャラクター・ライセンス範囲内。組み込み時にフッターへ「VOICE: 初音ミク」クレジットを追加します
